# Excel Data Analytics Project

## Overview
This project demonstrates a basic data analytics workflow using Microsoft Excel. The analysis includes loading a dataset, checking the data structure, creating pivot tables, and building charts to identify trends and insights. The project highlights practical Excel skills used in data analysis and reporting.

## Project Steps

### 1. Data Loading
- Imported dataset into Excel
- Verified successful loading of data
- Organized worksheet structure

### 2. Data Structure Checking
- Reviewed column names and data types
- Checked for missing or inconsistent values
- Validated dataset organization

### 3. Pivot Table Creation
- Created pivot tables to summarize data
- Performed grouping and aggregation
- Analyzed patterns and category-wise insights

### 4. Chart Creation
- Built charts based on pivot table analysis
- Visualized trends and comparisons
- Improved understanding of key findings

## How to Run
1. Download or open the Excel project file.
2. Open the dataset in Microsoft Excel.
3. Review the worksheets containing:
   - Raw data
   - Pivot tables
   - Charts and visualizations
4. Refresh pivot tables if needed using:
   **Data → Refresh All**
